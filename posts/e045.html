单词-后缀

词根 后缀
English 词根
发布日期:   2020-12-27
更新日期:   2021-01-08
文章字数:   436

名词

  • e
  • ty 常与i(连字符相连) = (i)ty = (e)ty = edy
  • ism 表抽象含义, ….主义
  • or= er = ar = ir = ur 名词后缀表人/物 , adj后缀, 表比较级
  • ory = ary = ery = ury 名词后缀/adj.
  • an = en 名词表人
  • al 表人, 物 rival 竞争者
  • im = um = em
  • ation = ate adj/v后缀 +ion n后缀
  • ism 抽象含义, 表……主义
  • ist 表人, 表形式主义
  • ness 表物, 学术, 学科性
  • ant = ent = and = end 名词/adj后缀 , 名词时表人, ant–ancy/ance ent–ency/ence
    • important = import 进口–物以稀为贵 重要
    • study student
  • ics 学术名词后缀
  • ive adj./n.
  • -dom 状态/领域 kingdom 王国 filmdom 电影界 officialdom 官场
  • ion 表物
  • itude correctitude 端正
  • it - = id adj/n后缀
  • ess 女性(雌性)名词后缀 actor 演员, actress 女演员, 把o顶掉 waiter 服务员, waiteress 女服员 tiger tigeress
  • ship 表抽象含义, friendship 友谊
  • ance = ence = ancy = ency (e和y可以互换) 偶尔可以动词化
  • anza
  • ch = c
  • able+ ary = abul + ary / ible + ity = ibel + ity = ibility
  • ette = et 表小 statuette 小雕塑
  • esque statuesque 雕像般的

形容词

  • al = el = il = ul
  • ous = ious = eous = uous
  • an
  • less 表否定
  • ed …的 主语是人
  • ible = able
  • able = ible = ile 表能…的
  • ative
  • logy 表….学
  • ing 形容词后缀 , 表令……的修饰物, 名词后缀
  • ic
  • ive adj/n 表人/物 d etective adj侦探的, n.侦探
  • ance

副词

  • ly(一般接在名词后为, 形容词, 接在形容词后, 为副词) friend+ ly = friendly 有好的, correct + ly = correctly 正确地

动词

  • ize = ise
  • e
  • ate = ite = ute adj后缀/v. 通常复合后缀(ation = ition = ution), 由ate 直接构成的单词可能直接名词
  • ish 造成….
  • fy 如果fy/y结尾, 加新后缀时, y通常变i再加, 后去掉 一旦名词化: i连字符+ fication = ification
  • it - = id adj/n后缀
